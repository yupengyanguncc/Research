# 躲在蚊子后面的大象

> 这几天你一直对我板着脸。你大概觉得，我应该问问你，是哪只蚊子叮咬了你。\\
>    <br />         ——约翰·沃尔夫冈·冯·歌德

没有人能够永远保持强大，自信和冷静．
所以，生气、恐惧、担忧、受伤等不愉快的情绪，
首先都是对 烦心事最正常的反应

我们需要知道一种情绪背后更深层次的原因．:thinking:

躲在蚊子后面的大象源于我们经历的所有负面体验．
